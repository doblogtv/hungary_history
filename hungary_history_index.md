# ハンガリー史 インデックス（出来事ベース）

## Ⅰ｜先史・カルパチア盆地の初期文化

**開始：紀元前6000年頃　終了：紀元前1000年頃**

主な出来事

* 紀元前6000年頃：カルパチア盆地で新石器文化成立
* 紀元前5500年頃：アルフェルド線帯文土器文化（Alföld Linear Pottery Culture）
* 紀元前4500年頃：ティサ文化（Tisza culture）形成
* 紀元前2800年頃：バーデン文化（Baden culture）
* 紀元前2000年頃：青銅器文化の拡大
